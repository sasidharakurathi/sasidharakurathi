<details open>
  <summary><b>🗓️ Typical Day Breakdown</b></summary>
  <br/>
  <p>

**Morning — Environment & Automation Setup**
> Kick off with PowerShell/Bash scripts to bootstrap dev environments. In *VirtuHire*, comprehensive `setup.ps1` and `setup.sh` scripts automate virtual-env creation, dependency installation, and service initialisation — making the project truly *clone-and-run* on any OS. *venvdrop* extends this philosophy further, bundling entire Python environments into self-verifying SHA-256-checked archives for offline/air-gapped deployment.

**Mid-morning — Core Logic & API Development**
> Design and iterate on FastAPI/Django backends. *AI-Interviewer* showcases real-time WebSocket signalling via Django Channels alongside an async FastAPI REST layer. *Project-T* demonstrates deep networking knowledge through a raw-TCP universal tunnel built on FastAPI. Rust code (Tauri commands) in *RoboSwift* and *Project-T* handles thread-safe state with `Mutex` guards for concurrent file and connection management.

**Afternoon — AI / ML Pipeline Work**
> Integrate and tune LLM pipelines. *Axon-File-Manager* features a multi-stage Neural Search engine (HyDE query expansion → BM25 keyword retrieval → Cross-Encoder re-ranking → Qdrant vector store). *repo-scribe* implements Smart Batching to feed large Git diffs plus Markdown docs into Gemini for RAG-enhanced commit messages. *glove_burger_detection* rounds out the day with YOLOv8 training pipelines and real-time video inference.

**Late afternoon — Cross-Platform Bundling & Testing**
> Package apps for distribution. Tauri CLI produces lightweight (~10 MB) cross-platform binaries for *RoboSwift* and *Project-T*. PyInstaller bundles *Axon-File-Manager* into a single executable. Flutter's `flutter build` compiles *route-memory* for Android/iOS. Unit and integration tests are run, `CHANGELOG.md` entries are drafted, and `TODO.txt` is updated with next-session priorities.

  </p>
</details>

<details open>
  <summary><b>✍️ Coding Habits</b></summary>
  <br/>
  <p>

- 📄 **Documentation-first**: Every project ships with a structured `README.md`, a versioned `CHANGELOG.md` and a `TODO.txt` for tracking in-flight work — consistently observed across prompt-polish, repo-scribe, venvdrop, and Axon-File-Manager.
- 🧩 **Modular Design**: Features are split into discrete, independently testable units. Axon-File-Manager's search pipeline is a prime example: each stage (HyDE expansion, BM25, re-ranking) lives in its own module with a clean interface. AI-Interviewer separates speech, vision, and session-management concerns into distinct async workers.
- 🔒 **Security-First Development**: API keys never touch a server — prompt-polish enforces a *Bring Your Own Key (BYOK)* model storing credentials in `chrome.storage.local`. venvdrop ships with SHA-256 integrity verification for every archive and supports fully air-gapped deployments with UDP-multicast discovery. ATS and AI-Interviewer handle biometric face-encoding caches with parallel-safe access patterns.
- ♻️ **DRY & Reusable Utilities**: Common patterns (async queues, recursive DOM helpers, RDP path-simplification) are extracted into shared utilities rather than duplicated, keeping individual modules lean and readable.

  </p>
</details>
